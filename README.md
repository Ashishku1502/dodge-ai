# Dodge AI

Welcome to **Dodge AI** - An impressive web application built with React and Vite.

## 🌐 Live Website
Visit the live application here: [https://dodge-ai-plum.vercel.app/](https://dodge-ai-plum.vercel.app/)

## 📋 About
Dodge AI is an innovative project designed to enhance gaming experiences through advanced artificial intelligence. It provides tools and resources for developers to integrate intelligent behaviors in gameplay, making games more engaging and dynamic.

## 🛠️ Tech Stack
- **Frontend**: React + Vite
- **Build Tool**: Vite with HMR (Hot Module Replacement)
- **Plugins**: 
  - [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) using [Oxc](https://oxc.rs)
  - [@vitejs/plugin-react-swc](https://github.com/vitejs/plugin-react-swc) using [SWC](https://swc.rs/)

## ✨ Features
- Advanced AI algorithms
- Easy integration
- User-friendly documentation
- Engaging and dynamic gameplay experience

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Build for production: `npm run build`

## 📝 ESLint Configuration
For production applications, we recommend using TypeScript with type-aware lint rules. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for more information.

## 🔧 React Compiler
The React Compiler is not enabled by default due to its impact on dev & build performance. To enable it, see [this documentation](https://react.dev/learn/react-compiler/installation).
